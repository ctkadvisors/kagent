    - Sound: every successful hotkey fires `sound.click()`; silent on timeout/no-match (avoids alert fatigue).

    | `o` | Open detail for current focus. Task focus → TaskDetail. Gateway focus → GatewayPage. Agent focus → toast "no Agent detail page in v0.2" (AgentDetail is future research). |
